# 基于tkinter的库存管理系统 python495

## 项目概述

本项目是一个基于Python语言的库存管理系统，使用tkinter库为用户提供图形用户界面（GUI）。通过该系统，用户可以进行账户注册、登录，以及对产品信息进行查看、添加、删除和修改等操作，同时支持查看出入库日志。

## 技术栈

- **开发语言**：Python
- **开发工具**：PyCharm/IDLE
- **前端库**：tkinter

## 功能模块

### 账户管理

- 登录：登录后方可进行后续操作，用户信息将永久化保存。

### 产品管理

- 查看产品：通过搜索框输入产品名称，查看产品详细信息。
- 添加产品：添加新产品信息，并永久化保存。
- 删除产品：从系统中删除产品信息，并永久化保存删除状态。
- 修改产品：对产品信息进行修改，并永久化保存更改。

### 日志管理

- 查看出库日志：查看产品的出库记录，信息永久化保存。
- 查看入库日志：查看产品的入库记录，信息永久化保存。

### 系统退出

- 退出系统：安全退出系统。

## 系统角色

- 系统管理员：负责注册、登录，以及对产品信息的管理。

## 运行环境

- Python运行环境（建议Python 3.x版本）
- PyCharm或IDLE
